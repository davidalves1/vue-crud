# vue-crud

> A project to test CRUD using the Vue.js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

#serve the api rest at localhost:3333/api
npm run api

# build for production with minification
npm run build
```
